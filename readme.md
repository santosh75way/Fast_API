# FastAPI Learning README (Junior-Friendly)

This README is a simple, practical guide to learn FastAPI step-by-step if you already know:
- basic Python
- backend basics (Node + Express)

---

## 1) What is FastAPI?
FastAPI is a modern Python web framework for building APIs.
It supports:
- async/await (ASGI)
- automatic request validation (Pydantic)
- automatic API docs (Swagger/OpenAPI)
- good performance
